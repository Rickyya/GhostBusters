Steps to run game.
1. Unzip .zip file.
2. Open Directory in Intellij
    a. File -> Open -> <path>
3. Go to src -> Ghostbusters -> ui -> GhostBustersGUI.scala
    a. if there is any error such as "setup scala SDK", click on setup SDK.
    b. click create -> download -> select scala 3.2.0.
4. If there are errors, right click on ghostbusters in project and then go to 'open module settings'
	a. project settings -> modules -> dependencies -> + -> JARs or Directories -> locate the scala-swing_3-3.0.0.0.jar -> check the checkbox 
	-> click apply and ok
5. Run.
